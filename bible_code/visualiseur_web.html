HTTP vs HTTPS au travers d'un proxy
HTTP — Proxy lit tout ✓
GET http://example.com/page HTTP/1.1
Host: example.com

↓ Le proxy voit :
GET /page HTTP/1.1
Contenu HTML lisible
{"data": "valeur"}
HTTPS — Proxy aveugle ✗
CONNECT example.com:443 HTTP/1.1

← Proxy répond :
200 Connection Established
